### Projects

**[IndiePulse](https://github.com/userbiznes/IndiePulse)** — "God Mode" for indie devs. Share App Store links, get instant revenue data on Dynamic Island. `Swift`

**[Midnight Concierge](https://github.com/userbiznes/airealestate)** — AI concierge for luxury real estate search. Full SwiftUI native app. `Swift`

**[Fragment Tracker](https://github.com/userbiznes/fragment-tracker)** — CLI that scrapes Telegram's Fragment marketplace for cheap short usernames. Deep scan bypasses the 500-listing API cap. `Python`

**[ZTCA Outreach](https://github.com/userbiznes/ZTCA-outreach)** — Local-first lead management and cold email tool. `TypeScript` `Next.js`

**[Sync](https://github.com/userbiznes/sync)** — Lock in with friends. Group focus sessions, squad accountability, leaderboards. `Swift`

**[PawPal](https://github.com/userbiznes/PawPal)** — Connecting dog owners with local walkers. MapKit + Google Sign-In. `Swift`
